# Cheeky Developer Preview

This folder contains development tools for previewing UI variants.

## Quick Start

Run the preview mode:
```bash
flutter run -t dev_preview/main_preview.dart
```

## Features

- **Live Theme Switching**: Use the dropdown to switch between all 10 UI variants in real-time
- **Hot Reload Compatible**: Make changes to any variant and see them instantly
- **Bypasses Onboarding**: Goes directly to the main swipe screen
- **Demo Data**: Uses placeholder profiles for testing

## UI Variants

1. **V1 Classic Tinder** - Classic card stack with slide left/right
2. **V2 Modern Glassblur** - Glassmorphism with blur effects
3. **V3 Dark Neon** - Dark mode with neon accents
4. **V4 Minimal Soft** - Clean minimalistic design
5. **V5 Luxury Gold Black** - Premium gold & black theme
6. **V6 Playful Pastel** - Fun pastel colors
7. **V7 High Contrast Red** - Bold aggressive red theme
8. **V8 Blue Corporate** - Professional Microsoft-like theme
9. **V9 Rounded Bubbles** - Soft bubble shapes
10. **V10 Card Stack 3D** - 3D perspective swipes

## Usage Notes

- The preview is for visual testing only
- Backend functionality (auth, messaging) is not active
- Use this for design reviews and A/B testing decisions
